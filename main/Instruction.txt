Здравствуйте!
Это инструкция к программе "Систематический поиск". Прочтите её перед использованием.
Программа выполняет поиск литературы в базе публикаций PubMed по всем возможным уникальным комбинациям слов, получаемых из таблицы.
Как пользоваться программой:
1. Создайте таблицу Excel с терминами для поиска.

Слова в таблице должны быть организованы следующим образом.
- Слова распределяются по колонкам
- Слова в каждой колонке синонимичны
- Для каждой группы синонимичных слов - своя колонка
Подписывать колонки или столбцы не надо.
Пример: тема "Влияние антибактериальной терапии на асептический некроз головки бедренной кости".
Ключевые слова: "Antibacterial therapy", "Antibiotics", "Antibacterial drugs", "Aseptic necrosis", "Avascular necrosis", "Ischemic necrosis", "Hip", "Femoral head", "Hip joint".
1 колонка: "Antibacterial therapy", "Antibiotics", "Antibacterial drugs".
2 колонка: "Aseptic necrosis", "Avascular necrosis", "Ischemic necrosis".
3 колонка: "Hip", "Femoral head", "Hip joint".
При использовании программы будут сформированы все возможные уникальные комбинации, в которых имеется по одному слову из каждой колонки.

2. Запустите программу, если ещё не сделали этого.
3. В случае запроса, введите свой настоящий адрес электронной почты.

Это важно, в случае чрезмерного использования PubMed может вас забанить, но сначала предупредит через почту.

4. Выберите "Выполнить поиск".
5. Выберите таблицу с ключевыми словами для поиска.
6. Выберите имя для таблицы вывода.
7. В консоль будут выведены комбинации слов, указанные комбинации будут использованы для поиска.
8. Идентификаторы статей будут выведены в консоль (их можно скопировать, выделив мышью и нажав комбинацию Ctrl+C).
После того, как все статьи будут найдены, начнётся скачивание их в выбранный Excel файл.
9. После скачивания программа вернётся в главное меню.
